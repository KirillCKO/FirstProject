Для корректной работы использовать последнюю версию ТГ бота и python версии 3.9.13

Варианты установки телебота:

pip install python-telegram-bot
python -m pip install pyTelegramBotAPI



Дополнительные библиотеки для работы функций:

--- `imagehash` для работы с хэш-данными файлов -> !pip install imagehash

--- `mtcnn` для детекциилиц на фото -> !pip install mtcnn
